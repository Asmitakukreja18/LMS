# E-Learning Website using HTML5, CSS3, Bootstrap5 and JavaScript 

![LMS Website](preview.png)
LMS — Online Learning Platform

LMS is a responsive e-learning website built using HTML, CSS, Bootstrap 5 and JavaScript.
It’s designed to give learners a clean, accessible experience to browse courses, register/login, add to cart and checkout.

Features

Fully responsive design (thanks to Bootstrap 5)

Course catalog with pricing and descriptions

Cart and checkout functionality (with basic JS logic)

User registration & login pages (frontend only)

Contact and about pages included

Designed for easy expansion (instructors, course management etc)

How to Run Locally

Clone this repo

git clone https://github.com/Asmitakukreja18/LMS.git  


Open the folder in VS Code (or your preferred editor).

Right-click index.html → Open with Live Server (or open it in browser).

Your website should display with full styles and functionality.
